Інструкція користування інструментом для планування занять у групі "Казка"

Цей інструмент призначений для створення та управління планами занять у групах дитсадків. Він дозволяє генерувати тижневі плани з урахуванням освітніх ліній та використовувати штучний інтелект для автоматичного заповнення тем і активностей.

Основні кроки роботи з інструментом:

  1. Налаштування періоду планування
    • • У розділі "Налаштуйте місяць плану" оберіть потрібний рік та місяць
    • • За замовчуванням обрано наступний місяць від поточної дати
  2. Генерація таблиць
    • • Натисніть "Створити порожні таблиці" для генерації тижневих планів
    • • Після створення таблиць кнопка змінить колір на помаранчевий
  3. Налаштування автозаповнення
    • • Відмітьте "Використовувати ШІ для автозаповнення" (доступно після створення таблиць)
    • • Введіть ваш Google Gemini API Key

      Google Gemini API Key надає доступ до можливостей штучного інтелекту Google для автоматичного заповнення планів занять.

      Отримання ключа:

      1. Відвідайте Google AI Studio
      2. Увійдіть у свій Google-акаунт
      3. Оберіть "Get API key" (зверху зліва)
      4. Створіть новий або використайте існуючий ключ
  4. Автоматичне заповнення
    • • "Заповнити теми тижнів" - генерує теми для кожного тижня
    • • "Заповнити теми днів" - створює теми для кожного дня
    • • "Заповнити таблиці активностями" - формує повний план занять
  5. Редагування та друк
    • • Використовуйте кнопки "Редагувати" та "Зберегти" для зміни змісту таблиць
    • • Натисніть "Друк" для отримання друкованої версії планів

Додаткова інформація:

  • • Всі дані автоматично зберігаються у браузері
  • • Для видалення всіх даних використовуйте кнопку "Очистити все"

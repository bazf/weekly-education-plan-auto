Інструкція користування інструментом для генерації тижневих планів навчання

Цей інструмент дозволяє генерувати тижневі плани навчання для дошкільнят, враховуючи обраний рік та місяць. Ви можете використовувати можливості штучного інтелекту для автоматичного заповнення тем тижнів, днів та навіть повного плану занять.

Основні кроки:

  1. Вибір року та місяця
    • • У блоці "Налаштування місяця, року та автозаповнення" оберіть потрібний рік
    • • Оберіть потрібний місяць
    • • За замовчуванням буде обрано наступний місяць від поточної дати
  2. Створення таблиць
    • • Натисніть кнопку "Створити порожні таблиці"
    • • Після створення таблиць кнопка змінить колір на помаранчевий та текст на "Перестворити порожні таблиці"
    • • Повторне натискання видалить існуючі таблиці та створить нові
  3. Активація автозаповнення (опціонально)
    • • Встановіть позначку "Використовувати ШІ для автозаповнення" (доступно тільки після створення таблиць)
    • • Введіть ваш Google Gemini API Key

      Google Gemini API Key - це спеціальний ключ доступу, який дозволяє нашому додатку використовувати штучний інтелект від Google для автоматичного заповнення планів.

      Щоб отримати цей ключ:

      1. Перейдіть на сайт Google AI Studio (aistudio.google.com)
      2. Увійдіть у свій Google-акаунт
      3. У верхньому лівому куті натисніть на значок профілю
      4. Оберіть "Get API key"
      5. Створіть новий ключ або використайте існуючий
      6. Скопіюйте ключ та вставте його в поле вводу на нашій сторінці

      Важливо знати:

      • • Ключ схожий на довгий набір букв і цифр
      • • Зберігайте ключ у безпечному місці і не діліться ним з іншими
      • • Ключ можна використовувати повторно
      • • Для користувачів з України сервіс безкоштовний
      • • Ключ зберігається виключно на Вашому пристрої і тільки Ви маєте до нього доступ
      • • При бажанні ключ можна видалити з додатку, використовуючи відповідну позначку біля кнопки "Очистити все"
    • • API ключ буде автоматично збережено
  4. Вибір моделі ШІ (опціонально)
    • • Встановіть позначку "Використати іншу модель ШІ"
    • • Оберіть бажану модель зі списку (від найшвидшої до найрозумнішої)
    • • За замовчуванням використовується оптимальна модель
  5. Автоматичне заповнення
    • • "Автозаповнення тем тижнів" - заповнює лише теми тижнів
    • • "Автозаповнення тем днів" - заповнює теми днів (доступно після заповнення тем тижнів)
    • • "Автозаповнення (повна таблиця)" - заповнює всю таблицю (доступно після заповнення всіх тем)

  • • Усі дані автоматично зберігаються у браузері

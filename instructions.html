Інструкція користування інструментом для генерації тижневих планів навчання для дошкільнят

Цей інструмент дозволяє генерувати тижневі плани навчання для дошкільнят, враховуючи обраний рік та місяць. Ви можете використовувати можливості штучного інтелекту для автоматичного заповнення тем тижнів, днів та навіть повного плану занять.

Основні кроки:

  1. Вибір року та місяця:
    • У блоці "Налаштування місяця, року, API-ключа та автозаповнення" оберіть потрібний рік з випадаючого списку "Оберіть рік".
    • Оберіть потрібний місяць з випадаючого списку "Оберіть місяць".
  2. Генерація таблиць:
    • Натисніть кнопку "Згенерувати таблиці". Після цього будуть створені таблиці для кожного робочого тижня обраного місяця.
  3. Введення API-ключа (для автозаповнення):
    • Для використання функцій автоматичного заповнення необхідно ввести ваш Google Gemini API Key у поле "Google Gemini API Key".
    • Натисніть кнопку "Зберегти ключ", щоб зберегти ваш ключ у локальному сховищі браузера.
  4. Використання інших моделей Штучного інтелекту:
    • Якщо ви хочете використовувати інші моделі Gemini, відмінні від встановленої за замовчуванням, встановіть позначку у чекбокс "Використати іншу модель ШІ".
    • Оберіть бажану модель з випадаючого списку "Оберіть модель".
    • Мовні моделі відсортовані від найшвидшої до найрозумнішої.
  5. Автозаповнення тем тижнів:
    • Після генерації таблиць та введення API-ключа кнопка "Автозаповнення тем тижнів" стане активною.
    • Натисніть цю кнопку, щоб автоматично згенерувати теми для кожного тижня.
  6. Автозаповнення тем днів:
    • Після заповнення тем тижнів кнопка "Автозаповнення тем днів" стане активною.
    • Натисніть цю кнопку, щоб автоматично згенерувати теми для кожного дня тижня, враховуючи теми тижнів.
  7. Повне автозаповнення:
    • Після заповнення тем тижнів та днів кнопка "Автозаповнення (повна таблиця)" стане активною.
    • Натисніть цю кнопку, щоб автоматично заповнити всі поля таблиць планами занять на кожен день, враховуючи теми тижнів та днів.
  8. Редагування таблиць:
    • Щоб відредагувати вміст таблиці вручну, натисніть кнопку "Редагувати" під потрібною таблицею.
    • Після внесення змін натисніть кнопку "Зберегти", щоб зберегти ваші зміни.
  9. Друк:
    • Натисніть кнопку "Друк", щоб відкрити вікно друку та роздрукувати ваші плани.
  10. Очистити все:
    • Натисніть кнопку "Очистити Все", щоб видалити всі згенеровані таблиці та введені дані (включаючи API-ключ) та перезавантажити сторінку.

Поради:

  • Переконайтеся, що ваш API-ключ є дійсним, щоб функції автозаповнення працювали коректно.
  • Перевіряйте згенерований контент та за потреби вносьте власні корективи.
  • Функція друку оптимізована для альбомної орієнтації сторінки A4.

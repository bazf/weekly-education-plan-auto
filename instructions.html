Інструкція користування інструментом для генерації тижневих планів навчання

Цей інструмент дозволяє генерувати тижневі плани навчання для дошкільнят, враховуючи обраний рік та місяць. Ви можете використовувати можливості штучного інтелекту для автоматичного заповнення тем тижнів, днів та навіть повного плану занять.

Основні кроки:

  1. Вибір року та місяця:
    • У блоці "Налаштування місяця, року та автозаповнення" оберіть потрібний рік
    • Оберіть потрібний місяць
  2. Генерація таблиць:
    • Натисніть кнопку "Згенерувати таблиці"
  3. Введення API-ключа (для автозаповнення):
    • Введіть ваш Google Gemini API Key
    • Натисніть "Зберегти ключ"
  4. Використання інших моделей ШІ:
    • Встановіть позначку "Використати іншу модель ШІ"
    • Оберіть бажану модель зі списку
    • Моделі відсортовані від найшвидшої до найрозумнішої
  5. Автозаповнення тем тижнів:
    • Натисніть "Автозаповнення тем тижнів"
  6. Автозаповнення тем днів:
    • Натисніть "Автозаповнення тем днів"
  7. Повне автозаповнення:
    • Натисніть "Автозаповнення (повна таблиця)"
  8. Редагування таблиць:
    • Використовуйте кнопки "Редагувати" та "Зберегти"
  9. Друк:
    • Натисніть кнопку "Друк" для друку планів

Поради:

  • Переконайтеся, що ваш API-ключ є дійсним
  • Перевіряйте згенерований контент
  • Функція друку оптимізована для альбомної орієнтації A4
